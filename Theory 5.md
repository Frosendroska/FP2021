##Натуральные числа (числа Чёрча)


Договоримся, что
```haskell
> 0 = \s z → z
> 1 = \s z → s z                  [fun fact: 1 = iff = id]
> 2 = \s z → s (s z)              [notice: не \s z → s s z]
> 3 = \s z → s (s (s z))
> 4 = \s z → s (s (s (s z)))
> 5 = \s z → s (s (s (s (s z))))
> ...
```


Определим операцию succ, делающую +1
```haskell
> succ 0 = 1
> succ 1 = 2
> ...
```

```haskell
> succ  = \n s z → n s (s z)
> succ' = \n s z → s (n s z)
```

Определим при помощи неё plus

Ответ:
```haskell
> plus = \n m s z → n s (m s z)
> plus' = \n m → n succ m
> plus'' = \n m → m succ n
```

```haskell
plus 2 3 =>
(\n m → n succ m) 2 3 =>
2 succ 3 =
(\s z → s (s z)) succ 3 =>
succ (succ 3) =>
5
```
